# Gozza
A module for post-processing tokenized text with a user-defined dictionary.

## Usage (WIP)
```
gazzzz --input "..." --dictionary "..." --output "..."
```

### Example
```
# input.txt
ฉัน|ขวัญ|หนี|ตี|ฝ่อ|ใจ|สลาย

# dictionary.txt
ตีฝ่อ
หลบลี้
คิดถึง

# output.txt
ฉัน|ขวัญ|หนี|ตีฝ่อ|ใจ|สลาย
```
